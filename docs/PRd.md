# [PRD] 수원 시립 미술관 웹사이트 리디자인: 인지 가독성 개선

## 1. 프로젝트 배경
현재 수원 시립 미술관 웹사이트는 아이콘 중심의 미니멀한 UI를 채택하고 있으나, 이는 디지털 숙련도가 낮은 고령층이나 장애인 등 정보 취약계층에게 심각한 이용 장벽을 형성하고 있습니다. 공공기관 웹사이트로서의 형평성을 확보하고, 모든 시민이 물리적 방문 전 온라인에서 전시 예약 및 정보 획득을 원활히 할 수 있도록 UI/UX를 전면 개선합니다.

---

## 2. 문제 정의 (The Core Problem)
* **추상적 아이콘의 한계:** '문 모양', '사람 모양' 등 텍스트 없는 아이콘은 사용자에게 '해석'의 고통을 주며, 특히 고령 사용자는 이를 기능 버튼이 아닌 단순 장식으로 오해함.
* **심리적 저항선 발생:** 조작 결과에 대한 확신이 부족하여 오클릭 시 "내가 실수했다"는 자책감을 느끼고 사이트 이용을 중단(이탈)함.
* **검색 접근성 부재:** 통합 검색창이 아이콘 뒤에 숨겨져 있어, 메뉴 탐색에 실패한 사용자의 마지막 구제책인 '검색' 기능이 제 역할을 수행하지 못함.

---

## 3. 핵심 UX 가설
> **"만약 우리가 주요 아이콘(로그인, 검색, 마이페이지 등) 옆에 명확한 한글 레이블을 병기하면, 사용자는 기능을 찾기 위한 망설임 없이 한 번에 정확한 메뉴로 진입할 것이며 인지 부하가 획득한 정보량 대비 현저히 낮아질 것이다."**

---

## 4. 타겟 페르소나
* **이름:** 박종식 (72세, 은퇴 교사)
* **IT 숙련도:** 하 (유튜브 시청은 가능하나 UI 은유 이해 불가)
* **방문 목적:** 손주와의 주말 전시 예약 및 경로 우대 혜택 확인
* **Pain Point:** 텍스트 없는 아이콘 앞에서 극심한 긴장을 느끼며, 잘못된 페이지 진입 시 공포감을 느껴 창을 닫아버림.

---

## 5. 핵심 요구사항 및 기능 명세

| 구분 | 요구사항 명칭 | 상세 기능 명세 (Feature Spec) | 우선순위 |
| :--- | :--- | :--- | :---: |
| **UI** | **헤더 레이블 병기** | 모든 유틸리티 아이콘(로그인, 마이페이지 등) 우측 또는 하단에 **14px 이상의 한글 레이블**을 반드시 노출함. | P0 |
| **Search** | **검색창 실체화** | 돋보기 아이콘을 누른 후 입력하는 방식이 아닌, **상시 노출된 검색 입력 상자(Search Box)** 형태로 배치함. | P0 |
| **Access** | **시각 피드백 강화** | 마우스 호버(Hover) 및 키보드 초점(Focus) 시 버튼 테두리 강조 및 색상 반전을 통해 **조작 가능 상태**임을 명확히 알림. | P1 |
| **Content** | **용어 순화** | 'My Page', 'Exhibition' 등 외래어/영문을 **'나의 예약', '전시 안내'** 등 직관적인 우리말로 교체함. | P1 |
| **Dev** | **접근성 속성 적용** | 모든 이미지와 버튼에 `alt` 태그 및 `aria-label`을 적용하여 스크린 리더 호환성을 100% 확보함. | P0 |

## 6. 성공 측정 지표 (Success Metrics)

* **TCT (Task Completion Time):** 고령 사용자의 메인 페이지 접속 후 '로그인 완료'까지의 시간이 기존 대비 **50% 이상 단축**.
* **Error Rate:** 로그인 시도 시 '회원가입' 버튼을 잘못 누르는 등의 **오클릭 발생률 30% 이하로 감소**.
* **Search Usage:** 검색창 실체화 이후, 메인 페이지에서의 **검색 기능 이용률 20% 이상 상승**.
* **Exit Rate:** 특정 메뉴 진입 전 망설임으로 인한 **메인 페이지 이탈률(Bounce Rate) 감소**.
